Tailwind CSS + Vanilla JS + Vite Starter
===========================

[](https://github.com/wdebusschere/tailwind-4-js-vite)

> A [Tailwind CSS](http://tailwindcss.com/) starter template for [Vite](https://github.com/vitejs/vite)



Installation
------------


```source-shell
npm install
```

Development
-----------


```source-shell
npm run dev
```

Build
-----


```source-shell
npm run build
```
